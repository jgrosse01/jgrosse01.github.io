Python

Python is an object-oriented language, like Java, and operates at a high level with built in data structures and dynamic user interactions. Python is often used for Rapid Application Development (RAD) because of it's compatibility with a console the user can interact with to test pieces of code.

Programmers often fall in love with Python simply because it does not require compilation between edit and run cycles. This lack of compiler time increases overall productivity of the Python coder. In addition, Python's error interpreter does not cause direct fault in running a program; rather, it raises an exception and in the case that the program cannot continue, it prints the stack trace, or a point by point map leading to the original location of the error in the code.

Python has been used to code several popular games in its time. Some of the familiar titles include Battlefield 2, Sid Meier's Civilization IV, Disney's Toontown, and EVE Online.

There are particular pros and cons to each language. Here are some pros and cons of Python:

Pros

  • Python has simple and lenient syntax
  • Python doesn't have a compile time therefore improving productivity
  • Simplification of class structures allow for easier referencing of other files

Cons

  • Graphics are more complex than some alternative languages
  • Though editing is faster, running Python code is much slower than other languages as it is read as it runs rather than compiled before it is run
  • Pythons easier syntax is a result of higher memory usage and overhead requirements
