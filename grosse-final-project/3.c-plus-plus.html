C++

C++ is a high-level, general-purpose computing language. It was created by Bjarne Stroustrup and was intended as an extension of the programming language C.

The '++' is derived from the idea of an incrementor used in many languages where you may use the '++' before or after a variable reference to increment it by one either before or after its use. C++ is therefore considered one up from C and contains its own library classes.

C++ has been standardized by the International Organization for Standardization (ISO) and was published by the ISO in and since 1998. A new version is released every three years; the current version is C++17 and the next planned version is C++20.

Many games made with C++ are often recognized and iconic such as Tetris, Pong, Flappy Bird, Counter Strike, Doom, and World of Warcraft. Electronic Arts (EA), a game development corporation, codes all of their games in an engine which was designed in house and utilizes C++. In addition, Microsoft codes all of their games in C++.

There are particular pros and cons to each language. Here are some pros and cons of C++:

Pros

  • Significantly faster than other languages due to its method of compiling code
  • Powerful graphics potential
  • Allows for greater line-by-line control of code

Cons

  • Very strict syntax
  • No automatic memory optimization
  • Graphics are very code intensive
